# Assets

Local large model assets (for example SD1.5 Core ML packages) live here.

- This directory is intentionally git-ignored to avoid pushing multi-GB binaries.
- Keep reproducible metadata and instructions in docs/configs; keep binary payloads local.

Expected optional layout:

- `assets/sd15_coreml/original/...`
- `assets/sd15_coreml/split_einsum/...`
